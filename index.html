LLM Visualizer RU

Интерактивная визуализация: токены → эмбеддинги → attention → выбор следующего токена.

1) Токенизация

2) Эмбеддинги (демо)

У каждого токена есть вектор. Здесь показана его упрощённая проекция.

3) Attention матрица

Насколько каждый токен “смотрит” на остальные.

4) Следующий токен

Легенда стадий

  1. Токенизация — разбиение текста на токены (кусочки слов/символов).
  2. Эмбеддинги — перевод токенов в числовые векторы (смысловые координаты).
  3. Attention — расчёт, какие токены важнее друг для друга в контексте.
  4. Next token — оценка вероятностей и выбор следующего токена.

Важно: это учебная визуализация (упрощённая), а не точная внутренняя трасса конкретной модели.
